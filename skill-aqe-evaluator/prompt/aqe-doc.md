## 典型已知失败模式（文档PPT方向）

### KF-DOC-001：信息过载
- **触发条件**：未在 Skill prompt 中设定每页信息密度上限
- **症状**：单页 PPT 有 6-8 条要点，字号全部 14px，密密麻麻
- **修复指令**：在 prompt 中加「每页核心信息点 ≤ 3 条，每条 ≤ 25 字，其余作为备注」

### KF-DOC-002：字体未授权
- **触发条件**：未在 Skill prompt 中指定字体
- **症状**：PPT 中出现苹方或微软雅黑（商用版权受限）
- **修复指令**：改用 Noto Sans SC 或思源黑体（完全开源）

### KF-DOC-003：跨页颜色不统一
- **触发条件**：AI 每页独立生成，未有全局色板约束
- **症状**：第1页蓝色调，第5页绿色调，第10页橙色调
- **修复指令**：在 Skill prompt 中提前锁定「全文主色 #2563EB，强调色 #F59E0B，背景 #F8FAFC」
