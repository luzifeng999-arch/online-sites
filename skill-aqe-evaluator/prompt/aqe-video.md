- L2 独立动效（≤3s）：LottieFiles优质案例
- L3 复杂视频（≤30s）：Remotion示例库

## 竞品对标
- CSS 微动效 → Framer Motion 示例库
- Lottie 动效 → LottieFiles 优质案例
- React 动效 → Remotion 示例库

## 前提检查

| # | 前提项 |
|---|--------|

## AQE-B 色彩美学专属检查（权重35%）

| 检查项 | 失分标准 |
|--------|---------|
| 缓动函数非线性：禁止使用 linear ease（匀速）作为默认动效，必须使用 ease-out / ease-in-out 或自定义贝塞尔 | 使用了 linear ease 作为主要动效 |
| 时长合理：UI微动效 200–400ms，过渡动效 300–500ms，无过快（<100ms 感知不到）或过慢（>1s 显卡顿） | 主要动效时长超出范围 |

## AQE-D 品牌一致性专属检查（权重35%）

| 检查项 | 失分标准 |
|--------|---------|
| 品牌调性匹配：动效速度/节奏与品牌性格一致（活泼品牌→弹性动效，严肃品牌→克制过渡） | 动效风格与品牌调性明显不符 |
| 跨次风格稳定：相同参数多次生成，动效节奏/风格无明显漂移 | 3次中有1次风格明显不同 |

## AQE-E 可访问性专属检查（强制，包含 E4）

| 检查项 | 失分标准 | BLOCK? |
|--------|---------|--------|
| 🔴 文字对比度（若有字幕/文字）≥ 4.5:1 | 对比度 < 4.5:1 | **BLOCK** |
| 响应 prefers-reduced-motion：系统设置「减少动态效果」时，动效停止或降级为简单淡入淡出 | 未响应系统动效设置 | 打回 |
| 无快速闪烁：不含频率 > 3Hz 的闪烁内容（防止光敏性癫痫风险） | 存在 >3Hz 的闪烁 | 打回 |

## 典型已知失败模式（视频动效方向）

### KF-VID-001：匀速动效
- **症状**：所有元素使用 `animation-timing-function: linear`
- **修复指令**：改为 `ease-out`（入场动效）或 `ease-in-out`（过渡动效），或用 `cubic-bezier(0.4, 0, 0.2, 1)` 自定义
