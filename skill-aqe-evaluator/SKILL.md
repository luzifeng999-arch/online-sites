# AQE Evaluator · Skill 评测专家

## 简介

对任意**设计类 Skill** 执行官方 AQE（Aesthetic Quality Evaluation）五维度美学评测，  
输出符合官方 Gate 2 标准的评测报告（精美 HTML 格式）。

本 Skill 基于：[设计侧 Skill 美学与体验准入准出标准](https://huwenji1215-ai.github.io/skill-standards/skill-design-standard.html)

---

## 能做什么

1. **识别 Skill 方向**（UI生成 / 图片生成 / 视频动效 / 文档PPT / 评测）
2. **执行 AQE 五维度打分**（视觉层级 / 色彩美学 / 版式规范 / 品牌一致性 / 可访问性）
3. **准入 Checklist 核查**（按五大方向专属清单逐项判断）
4. **分析被评测 Skill 的输出样本**（HTML / 截图 / 文档 均支持）
5. **给出 PASS / WATCH / FAIL / BLOCK 判定结论**
6. **输出完整精美 HTML 评测报告**（可直接发给设计 TL 或工程团队）

---

## 适用场景

- Skill 上线前做 Gate 2 自查
- 对已有 Skill 进行美学质量摸底
- 对比多个 Skill 的美学质量
- 建立团队级 Skill 质量基线

## 输入格式

提供以下任意组合（信息越完整，评测越准确）：

### 必须提供（至少一项）
| 信息类型 | 说明 |
|----------|------|
| Skill 名称 + 功能描述 | 说明该 Skill 做什么、输出什么 |
| Skill 的 SKILL.md / 文档 | 直接粘贴 Skill 的规范文档 |
| Skill 的输出样本 | HTML 代码 / 截图 / 文档内容 |

### 选填（提高评测精度）
| 信息类型 | 说明 |
|----------|------|
| 方向标注 | 手动指定方向（如"这是 UI 生成 Skill"）|
| 竞品对比截图 | 用于 D 维度（品牌一致性）评估 |
| 多次输出样本 | 用于 D1（跨次风格一致性）评估 |
| 完整 HTML / 源码 | 用于 E1（对比度）精确计算 |

---

## 输出

**完整自包含 HTML 文件**，包含：

- Skill 信息概览（名称 / 方向 / 评测时间）
- 综合分 + 判定徽章（PASS / WATCH / FAIL / BLOCK）
- SVG 雷达图（5 维度可视化）
- 五维度详情（各维度得分 + 失分原因 + 可执行修复建议）
- 方向专属 Checklist（逐项 ✓/✗，🔴 标注强制 BLOCK 项）
- Known Failures 摘录（典型失败模式）
- 修复优先级排序

---

## 评分逻辑概述

### AQE 五维度（各 1–10 分）

| 维度 | 说明 | 强制项 |
|------|------|--------|
| A · 视觉层级 | 字号梯度、视觉动线、信息层次是否清晰 | 否 |
| B · 色彩美学 | 主色数量、饱和度、色彩情感与场景匹配度 | 否 |
| C · 版式规范 | 4/8px 间距体系、对齐、留白 | 否 |
| D · 品牌一致 | 多次调用风格稳定性、品牌调性匹配 | 否 |
| E · 可访问性 | 对比度≥4.5:1、颜色非唯一信息载体 | **E1、E2 < 5.0 → BLOCK** |

### 综合分计算

综合分 = Σ（各维度分 × 对应方向权重）

| 方向 | A | B | C | D | E |
|------|---|---|---|---|---|
| UI生成 | 30% | 25% | 25% | 10% | 10% |
| 图片生成 | 15% | 45% | 10% | 20% | 10% |
| 视频动效 | 10% | 35% | 10% | 35% | 10% |
| 文档PPT | 25% | 15% | 35% | 15% | 10% |
| 评测类 | 0% | 25% | 0% | 45% | 30% |

### 判定规则

| 判定 | 条件 |
|------|------|
| ✅ PASS | 综合 ≥ 7.5，且无维度 < 6.0 |
| 👀 WATCH | 综合 7.0–7.4，无维度 < 6.0 |
| ❌ FAIL | 综合 < 7.0，或任意维度 < 6.0 |
| 🚫 BLOCK | AQE-E 子项 E1 或 E2 < 5.0（强制阻断） |

---

## 局限说明

- 本 Skill 为 **AI 辅助评测**（LLM-as-Judge），不能完全替代人工 Gate 2
- 若无具体输出样本，评测将基于描述推断，准确率约 70–80%
- 视频/动效方向建议提供帧截图，AI 无法直接处理视频文件
- E1 对比度精确计算需要 HTML 源码或截图，纯文字描述只能给出近似估算
